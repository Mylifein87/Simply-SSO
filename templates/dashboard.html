This project was written in Python using the Flask framework to provide a simple lightweight Single-On web app I decided to get my hands dirty by implementing Auth0 as my identity service to provideAuthentication and Authorization to my application. Upon successful authentication this project simulates an End User accessing a company resource. Continuing on with my learning I also decided to use Docker to deploy and run this application which resulted in a few hours of errors and troubleshooting as a beginner with this technology but overall was a fun project and learning experience.
